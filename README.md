# ☕ Alex Cafè - Sito Web

Sito web ufficiale di **Alex Cafè**, sviluppato con Next.js, TypeScript e Tailwind CSS.

## 🚀 Tecnologie utilizzate

- [Next.js 16](https://nextjs.org/) — Framework React per applicazioni web
- [React 19](https://react.dev/) — Libreria UI
- [TypeScript](https://www.typescriptlang.org/) — Tipizzazione statica
- [Tailwind CSS](https://tailwindcss.com/) — Framework CSS utility-first
- [ESLint](https://eslint.org/) — Linting del codice

## 📁 Struttura del progetto

```
alex-cafe/
├── src/
│   ├── app/          # Pagine (App Router Next.js)
│   ├── components/   # Componenti React riutilizzabili
│   ├── data/         # Dati statici
│   └── styles/       # Stili globali
├── public/           # Asset statici (immagini, icone)
├── next.config.ts
├── tailwind.config
└── package.json
```

## ⚙️ Installazione e avvio

```bash
# Clona la repository
git clone https://github.com/reda10nafil/Alex-caf-.git

# Entra nella cartella del progetto
cd Alex-caf-/alex-cafe

# Installa le dipendenze
npm install

# Avvia il server di sviluppo
npm run dev
```

Il sito sarà disponibile su [http://localhost:3000](http://localhost:3000)

## 📦 Script disponibili

| Comando | Descrizione |
|---|---|
| `npm run dev` | Avvia il server di sviluppo |
| `npm run build` | Build di produzione |
| `npm run start` | Avvia il server in produzione |
| `npm run lint` | Controlla il codice con ESLint |
