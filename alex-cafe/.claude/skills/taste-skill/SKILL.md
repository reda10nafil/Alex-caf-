# taste-skill — Visual Art Direction

## Role
Shape the mood and aesthetic identity of Alex Cafe to feel warm, editorial, tactile, urban, curated, photographic, and authentic.

## When This Skill Applies
- Color palette decisions
- Typography pairing choices
- Image treatment and photography direction
- Card and component aesthetic
- Section backgrounds and textures
- Overall page mood and atmosphere
- Promotional styling for offers/events

## Mood Direction
Alex Cafe should feel like a **modern neighborhood cafe** — not a generic restaurant template, not a SaaS landing page, not a luxury pretension.

### Reference mood board:
- Modern Italian cafe
- Boutique bistro in Turin
- Industrial-chic neighborhood venue
- Editorial hospitality magazine

### Feeling words:
- Warm
- Welcoming
- Urban
- Contemporary
- Authentic
- Curated
- Tactile
- Premium but human

## Color Palette

### Primary palette (warm neutrals)
```
Cream:       #FAF6F1  — primary background
Cream Dark:  #F0EAE0  — secondary background, alternating sections
Stone:       #D4C9B8  — borders, dividers, subtle elements
Stone Dark:  #B8A994  — hover states on neutral elements
Espresso:    #3C2A1A  — primary text, dark sections, header
Charcoal:    #2D2926  — deepest text, footer background
Warm Gray:   #6B5E52  — secondary text, descriptions
```

### Accent palette
```
Accent (Burnt Orange):  #C4622D  — CTAs, highlights, active states
Accent Light:           #D4844E  — hover states on accent
Accent Dark:            #A84E1F  — pressed states
```

### Secondary accent (for differentiation)
```
Sage Green:     #7B8C5E  — offers tags, student section, freshness cues
Sage Light:     #96A876  — hover states
```

### Rules:
- Accent color appears in CTAs, key highlights, active states — never everywhere
- Background alternates between cream and cream-dark for section rhythm
- Dark sections (espresso/charcoal) used sparingly: hero, footer, special highlights
- Never use blue/purple AI gradients or neon colors

## Typography

### Display font: Playfair Display
- Used for: Hero headlines, section headings, event titles
- Weight: Bold (700) for headlines, Semibold (600) for subheadings
- Creates editorial, sophisticated feel

### Body font: Inter
- Used for: Body text, menu items, form labels, buttons, navigation
- Weight: Regular (400) for body, Medium (500) for emphasis, Semibold (600) for buttons
- Highly readable on mobile, clean and modern

### Rules:
- Display font only for headings — never for body text
- Strong contrast between display personality and interface clarity
- Minimum body text: 16px (--text-base)
- Line height for body: 1.5 minimum
- Letter spacing: slightly wider for uppercase labels (--tracking-wide to --tracking-wider)

## Image Treatment
- Use real photos of the cafe whenever possible
- Warm, slightly desaturated tones — not over-processed
- Hero images: atmospheric, establishing the space
- Food/drink: close-up, natural light, appetizing
- People: candid, authentic, welcoming
- No stock-photo feel — everything should feel real

## Card Aesthetic
- White background with subtle warm shadow
- Border radius: --radius-lg (12px)
- Image on top (16:9 or 4:3 ratio)
- Content below with generous padding
- Subtle hover elevation change (shadow-md → shadow-lg)

## Anti-Patterns (never produce)
- Shiny corporate SaaS aesthetics
- Purple-blue AI gradients
- Over-neon nightclub styling
- Fake luxury black-and-gold clichés
- Childish cafe branding
- Generic themeforest restaurant layouts
- Overdecorated cafe clichés
